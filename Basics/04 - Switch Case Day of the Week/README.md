# Switch Case - Day of the Week

## 📌 Platform

TakeUForward (TUF)

## 🟢 Difficulty

Easy

## 🏷️ Tags

Switch Statement, Conditional Statements, Basics

---

## 💡 Approach

The solution uses a `switch` statement to map each day number to its corresponding day of the week.

* `1` → Monday
* `2` → Tuesday
* `3` → Wednesday
* `4` → Thursday
* `5` → Friday
* `6` → Saturday
* `7` → Sunday
* Any other value → **Invalid**

The `default` case handles invalid inputs outside the range `1` to `7`.

## ✅ Key Takeaway

This problem demonstrates how a `switch` statement can be used as a clean and readable alternative to multiple `if-else` conditions when comparing a single variable against fixed values. The `default` case ensures invalid inputs are handled appropriately.
