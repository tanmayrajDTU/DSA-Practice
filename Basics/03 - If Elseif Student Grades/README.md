## 💡 Approach

The solution evaluates the student's marks using an `if-else if-else` ladder in descending order.

* If `marks >= 90`, print **Grade A**.
* Else if `marks >= 70`, print **Grade B**.
* Else if `marks >= 50`, print **Grade C**.
* Else if `marks >= 35`, print **Grade D**.
* Otherwise, print **Fail**.

Checking the highest grade first ensures the correct grade is assigned.

This problem demonstrates how to handle multiple conditions using an `if-else if` ladder. The order of conditions is important, as the first matching condition determines the output.
